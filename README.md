# Прототип админ-панели для службы доставки

Прототип веб-интерфейса для оператора службы доставки. Разработан в рамках дипломной работы "Разработка чат-бота для службы доставки".

## Функционал

- Авторизация (логин: admin, пароль: 123)
- Просмотр списка заказов
- Фильтрация заказов по статусу
- Просмотр деталей заказа
- Изменение статуса заказа
- Назначение курьера
- Выход из системы

## Страницы

| Страница | Файл | Описание |
|----------|------|----------|
| Вход | index.html | Форма авторизации |
| Список заказов | orders.html | Все заказы, фильтр по статусу |
| Карточка заказа | order-detail.html | Детали заказа, редактирование |

## Технологии

- HTML5
- CSS3 (адаптивная вёрстка)
- JavaScript (имитация данных, логика)

## Запуск (локально)

1. Скачайте все файлы в одну папку
2. Откройте index.html в браузере
3. Логин: admin, пароль: 123
